    Goland

    快捷键

    跳转

    • 指定行 cmd + L

    • 指定文件 shift + cmd + O

    • 返回光标上一个位置 cmd + opt + 方向键

    查询

    • 函数列表 cmd + f12

    • 当前文件查询 cmd + F

    • 当前文件替换 cmd + R

    • 所有文件查询 shift + cmd + F

    • 所有文件替换 shift + cmd + R

    选择

    • 选中下一个相同内容 ctrl + G
    • 选中所有相同内容 ctrl + G
    • 展开所有代码块 shift + cmd + =
    • 折叠所有代码块 shift + cmd + -
    • 折叠或展开当前代码块 cmd + .

    修改

    • 复制当前行内容到下一行 cmd + D
    • 删除当前行 cmd + delete
    • 格式化代码 option + cmd + L
    • 在当前行下一行增加空行 shift + enter
    • 在当前行上一行增加空行 option + cmd + enter
